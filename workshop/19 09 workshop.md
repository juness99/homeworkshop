2. ##### 다음 태그에서 잘못된 부분을 찾으시오.

```html
<!DOCTYPE html>

</head>
<body>
<img src="http://www.google.com/" alt="GOOGLE">
</body>
```

```
img태그에는 href라는 태그가 없다 그러므로 src를 사용해야한다.
```
